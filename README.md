## Technologies Used

- Next.js (React)
- Tailwind CSS
- Redux
- Firebase
- Cloud Firestore
- NextAuth
- Webhooks
- Stripe Checkout
- Fake Store API

# Installation Steps

## Using npm

Run commands

1. `npm install`

2. `npm run dev`

## Or using yarn

Run commands

1. `npm install --global yarn`

2. `yarn install`

3. `yarn run dev`
